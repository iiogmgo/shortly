## Developement/Testing Env

### Developing Language

- Python 3.7

### Framework

- Flask

## Setup

1. install requirements

   `pip install -r requirements.txt`
   
2. init db

   `python db.py db init`
   
   `python db.py db migrate`
   
   `python db.py db upgrade`
   
3. run flask

   `flask run`
